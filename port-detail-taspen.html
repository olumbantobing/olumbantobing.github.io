Project Management System

Project information

  • Category: Web Development
  • Client: IT Department PT Taspen (Persero)
  • Project date: Jul 2022 - Aug 2022
  • Project URL: Confidential

As a full-stack developer, I developed a web-based Project Management System to manage project requests received by the Information Technology Division of PT Taspen (Persero) from other divisions and from the division itself. The application can store new project data, add documents, and project workflows that are organized according to the role and position of the active project. The development of this project is built with Laravel on a local database and the implementation will be integrated by staff for the company's internal database.
